# Json Storage Application

This is an example of how to **store** data in JSON format.

Dependencies:

* Ejs
* Morgan
* Express
* uuid

Devdependencies:

* Nodemon.

> To install the application: 

`npm install`

> To _start_ the application:

`npm run dev`
